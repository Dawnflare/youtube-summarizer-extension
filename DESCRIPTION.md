# Chrome Web Store Description

Instantly summarize any YouTube video with the power of Google Gemini — no copy-pasting required.

YouTube Summarizer adds a clean "✨ Summarize with Gemini" button directly to YouTube video pages. One click opens Google Gemini in a new tab with the video URL pre-filled and the prompt auto-submitted, so you get a concise AI-powered summary in seconds — completely hands-free.

🎯 HOW IT WORKS
1. Visit any YouTube video page
2. Click the "✨ Summarize with Gemini" button (appears below the video title)
3. Gemini opens in a new tab and automatically generates a summary

⚡ KEY FEATURES
• One-click summarization — no manual copy-pasting of URLs or prompts
• Auto-submit — the prompt is filled and sent automatically, so the summary starts generating the moment the tab opens
• Works seamlessly with YouTube's single-page app navigation — the button appears reliably as you browse between videos
• Lightweight and fast — no background processes, no bloat
• Minimal permissions — only requests access to YouTube and Gemini pages
• Open source — fully transparent code on GitHub

🔒 PRIVACY FIRST
• No data collection — this extension does not collect, store, or transmit any personal data
• No tracking or analytics
• No third-party servers — your video URL goes directly to Google Gemini
• The only data stored locally is your logging preference setting

💡 PERFECT FOR
• Students researching topics covered in long lecture videos
• Professionals who need quick takeaways from conference talks and webinars
• Content creators scouting videos for inspiration
• Anyone who wants to decide if a video is worth watching before committing

⚙️ SETTINGS
• Configurable verbose console logging for developers and troubleshooting
